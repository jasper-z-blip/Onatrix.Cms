# Onatrix.Cms
Onatrix CMS – Umbraco-projekt

Beskrivning:
Detta är mitt Umbraco-projekt som jag byggt baserat på designfilen i Figma.  
Jag har försökt följa strukturen i designen så nära som möjligt och fokuserat på att få sidorna fungerande i Umbraco med Block List och en grundläggande redaktörsvänlig struktur.
Jag la tyvärr lite för mycket tid i början på att arbeta i både VS Code och Visual Studio innan jag började med själva Umbraco-delen och sen även Azure. Det gjorde att jag fick tidsbrist mot slutet och därför hann jag inte bygga klart alla block så som jag hade planerat. Jag trodde inte att Umbraco och Azure skulle ta så lång tid som det gjorde.

Tekniker:
- Umbraco CMS  
- ASP.NET Core  
- Block List Editor  
- SQLite  
- Azure App Service  
- GitHub
